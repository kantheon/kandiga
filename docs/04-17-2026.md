### 09:37 — 🎯 STATIC FIXED — DiT rotary cos/sin layout bug

**Root cause:** Mainline transformers 5.5 introduced a bug in `Qwen2_5OmniDiTRotaryEmbedding`. The module builds cos/sin with `torch.cat((freqs, freqs), dim=-1)` giving split-half layout `[f0..f_{n/2}, f0..f_{n/2}]`. But `rotate_half_codec` (same file) pairs neighbors `(x0,x1),(x2,x3),...` — only matches the INTERLEAVED layout `[f0,f0,f1,f1,...]`. The preview branch (`v4.51.3-Qwen2.5-Omni-preview` that Qwen's model card explicitly recommends) uses the correct `torch.stack((freqs,freqs), dim=-1).reshape(...)` for interleaved. Our MLX port had copied the buggy mainline layout.

**Why I missed it:** Line-by-line diff of our MLX code vs mainline showed "identical" — because we were both buggy in the same way. Required diffing against the preview branch to see it.

**The one-line fix** in `mlx_omni/token2wav_mlx.py:711`:
```python
# Before (buggy, matches mainline 5.5):
emb = mx.concatenate([freqs, freqs], axis=-1)
# After (matches preview 4.52, what rotate_half expects):
emb = mx.stack([freqs, freqs], axis=-1).reshape(*freqs.shape[:-1], -1)
```

**User verdict after fix:** "finally. CLEAR." — hiss/static gone.

**Peak RMS/amplitude changes after fix** (same prompts, same codec tokens, same everything else):
- Chelsie: peak 0.44 → **0.66**, RMS 0.061 → **0.092**
- Ethan: peak 0.84 → **0.82**, RMS 0.047 → **0.082**

Louder + coherent output = attention scores finally correct.

**How we found it:** User spotted the HF model card's "pip install git+@v4.51.3-Qwen2.5-Omni-preview" recommendation. Our mainline transformers is 5.5.0 — several majors ahead. Installed preview in isolated venv, confirmed input_ids byte-identical (not the bug). Diffed modeling_qwen2_5_omni.py preview (4644 lines) vs mainline (3988 lines) via research agent — identified the rotary layout regression as the likely root cause.

**Commit:** personaplex-mlx `3163cb4` — "FIX STATIC: DiT rotary interleaved cos/sin layout"

**Not just our bug:** every downstream user of mainline transformers 5.5's `Qwen2_5OmniDiTRotaryEmbedding` will have this. Should file an upstream issue / PR to HF transformers.

Worth shipping: MLX Qwen2.5-Omni-7B pipeline NOW works at 3.34 GB peak, 2.81s first-audio (memopt streaming), and clean audio. End-to-end real-time voice AI on Apple Silicon is done.
